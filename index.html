Project Webserver A is for Awesome

The goal of this project is to replicate Dr. Wang et al's expression data and make our data accessible and searchable on this website. Please view "Analysis Summary" for more details about our work.

About the Navigation Bar:

Project Webserver: The homepage

Analysis Summary: A detailed summary of our work

Data Dropdown:

  • Expression Levels: Query a gene and select from genome, trim status and pipeline options in order to extract FPKM values for pipeline 5, counts for pipeline 6. Two buttons available for "Gene Information" and "Graph Plotly."
    • "Gene Information" button: View information on queried gene.
    • "Graph Plotly" button: View a Plotly bar chart for queried gene, genome, trim status and pipeline.
  • Differential Expression: Query a gene and select from genome, trim status and pipeline options in order to extract logFC values, p values, and whether or not the query is differentially expressed.
  • Volcano Plot with R Shiny: Enter logFC and p values and choose genome, pipeline and trim status to form a volcano plot with R Shiny. Other options include only visualising upregulated genes or both downregulated and upregulated genes. The user could also enter the amount of genes to display. A value of 100 for instance would display the top 100 genes that fall within greater than 1.5 or less than -1.5 for the logFC and less than 0.05 for the p value. Also, the user could choose to only display the selected range.
  • Visualise Mapped Data: Query a gene and select from genome, trim status and pipeline options in order to visualise the gene on the UCSC Genome Browser.
  • View FastQC Files: View quality reports on fastq sequences per rat.
